**@codluv/versionguard** is a TypeScript documentation toolkit that performs a single AST traversal of your project and produces API docs, OpenAPI specs, executable doctests, and AI context files in one pass.

## Features

- Full TypeScript support with TSDoc extraction
- Auto-generated API reference from source code
- Executable `@example` blocks as doctests
- AI-ready context files from a single build pass

## Installation

```bash
npm install -D @codluv/versionguard
```

## Quick Example

```typescript
const feedbackResult = getVersionFeedback('1.2.3', config, '1.2.2');
console.log(feedbackResult.valid);
```
